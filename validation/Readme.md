# Validation Plots

This folder contains the canonical evidence plots for the three empirical experiments in `08_empirical/Results.md`.

These are generated by `generate_validation_plots.py` in the repo root.

---

## Files

| File | Experiment | What it shows |
|------|-----------|---------------|
| `fxso_validation_stress_thickness.png` | Exp 1 — Stress Test | Thickness over time for motion=0.06 and motion=0.0 (kill test). Marks t=300 (random shock) and t=600 (rotation kick). Shows partial invariance, not glide. |
| `fxso_validation_motion_sweep_raw.png` | Exp 2 — Motion Sweep | V_circ and thickness vs θ, raw per-seed values. Flat V_circ line confirms no phase transition. |
| `fxso_validation_mhat_regime.png` | Exp 4 — Mexican Hat | Four-panel: trajectory overlay, final state scatter, radius histogram, angular histogram. Shows orbital coherence — agents share a stable annular trajectory over time, but remain angularly concentrated at any instant (phase-locked). **Orbital Coherence, not Elastic.** |

---

## To regenerate

```bash
python generate_validation_plots.py
```

Requires numpy and matplotlib. Runs all three empirical scripts and saves plots here.

---

## What these plots confirm

- **Stress test:** rotation produces moderate disruption (+0.118), not a clean glide. Partial invariance.
- **Motion sweep:** V_circ flat across all θ. Critical velocity hypothesis falsified.
- **Mexican Hat:** trajectory = clean ring. Final state = tight angular cluster. V_circ = 0.003 means phase-locked, not uniform. **Orbital Coherence, not Elastic.**
